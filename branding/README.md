# Lumos Shared Logo Pack

Shared identity for both projects:
- `lumos-mark.svg`: square icon/mark
- `lumos-logo-app.svg`: horizontal lockup for the mobile app
- `lumos-logo-agent.svg`: horizontal lockup for the agent

Color system:
- `#19C7B4` / `#0E9F8D`: network/routing signal
- `#FFD57A` / `#F59E0B`: power/energy bolt
- `#0B1220` / `#0F172A`: dark base

Usage:
- Keep clear space around logos equal to the bolt height.
- Prefer the mark (`lumos-mark.svg`) for app icons and compact UI.
- Prefer horizontal lockups in docs, dashboards, and release notes.
