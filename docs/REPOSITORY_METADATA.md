### Reddit/Forum Introduction
```
Hey everyone! I built Lumos because I was tired of paying for proprietary software that locks basic remote-control features behind subscriptions.

Lumos is a LAN-first power-control stack:
- Android app for discovering and controlling devices
- Go agent for Windows/Linux with Wake-on-LAN and power commands
- Token-based auth with fine-grained permissions
- Works over VPN (tested with Tailscale)
- 100% open source, no cloud required

Perfect for home labs, network admins, or anyone who wants direct control of their devices without vendor lock-in.

GitHub: https://github.com/coff33ninja/lumos
```
