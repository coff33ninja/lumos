# Lumos Wiki

Lumos is a LAN-first remote power-control platform composed of:
- `lumos-agent` (Go, Windows/Linux)
- `lumos_app` (Flutter Android client)

Use this wiki as the operator-friendly reference for architecture, setup, security, and release operations.
